# 🏋️‍♂️ AI Health & Fitness Plan Generator

An AI-powered health assistant that generates **personalized diet plans and workout routines** based on user inputs such as age, weight, height, activity level, dietary preference, and fitness goals.

The application uses **multi-agent AI architecture** to combine nutrition planning, workout generation, and lifestyle guidance into a **complete health strategy**.

---

# 🚀 Features

* 🧠 AI-generated personalized **diet plans**
* 💪 Customized **workout routines**
* 🥗 Supports multiple diets (Keto, Vegetarian, Low Carb, Balanced)
* 🎯 Fitness goal based planning (Weight Loss, Muscle Gain, Endurance, Flexibility)
* 🌐 Web search support using **DuckDuckGo**
* 📊 Holistic health strategy combining diet and fitness
* 🖥️ Interactive **Streamlit UI**
* 📋 Nutritional breakdown and meal suggestions
* ⚡ Real-time AI response using **Gemini**

---

# 🛠️ Technologies Used

* **Python**
* **Streamlit** – Web application interface
* **Google Gemini AI** – Health plan generation
* **Agno Agent Framework** – Multi-agent orchestration
* **DuckDuckGo Tools** – Web search for additional health insights

---

# 🧠 AI Agent Architecture

The system uses **three intelligent agents** working together.

## 1️⃣ Dietary Planner Agent

Responsibilities:

* Generates personalized meal plans
* Includes breakfast, lunch, dinner, and snacks
* Considers dietary preferences
* Provides nutrition breakdown
* Suggests meal preparation tips

---

## 2️⃣ Fitness Trainer Agent

Responsibilities:

* Generates customized workout routines
* Includes warm-ups, exercises, and cool-downs
* Adjusts plans for fitness levels
* Provides safety and injury prevention tips
* Suggests progress tracking methods

---

## 3️⃣ Team Lead Agent

Responsibilities:

* Combines diet and workout plans
* Creates a holistic health strategy
* Aligns nutrition with workouts
* Provides lifestyle and motivation tips

# 📦 Installation

## 1️⃣ Clone the Repository

git clone https://github.com/yourusername/ai-health-fitness-planner.git

cd ai-health-fitness-planner

---

## 2️⃣ Install Dependencies

pip install streamlit agno google-generativeai duckduckgo-search

---

# 🔑 API Key Setup

You need a **Google Gemini API key**.

Replace in the code:

GOOGLE_API_KEY = "your_google_api_key"

---

# ▶️ Running the Application

Run the Streamlit app:

streamlit run app.py

Then open in browser:

http://localhost:8501
